====================================================
Nested dictionaries
====================================================

This uses nested dictionaries to display chosen information.

----

References
------------------------------

#. https://anvil.works/docs/media/image-manipulation

----

Design
---------

| Use an XY panel for easy resizing and placement of fields.
| Use 4 option buttons to choose the information to display.
| Use 6 label fields to display the information type and content.
| Add images to the assets to display with the text information.
